01 — The Signal

Three patterns predict every dropout.

Recency drift

Login gaps compound. A student missing one week is recoverable. Missing three weeks is almost always a permanent exit.

Model weight

Completion stall

Progress doesn't stop suddenly. It slows — then freezes. Average completion below 40% is the critical threshold.

Model weight

Assessment silence

Skipping quizzes is a leading indicator, not a lagging one. Students who disengage from assessments first, drop out second.

04 — System Design

Three layers. One contract.

1

Data Layer

Synthetic telemetry generated with NumPy — mimicking real EdTech signals including seasonal churn and ghost-student edge cases.

generator.py → students.csv
2

Intelligence Layer

Random Forest classifier trained on engineered features. Recall ≥ 0.80 as the primary metric — catching dropouts matters more than eliminating false alarms.

model.py → model.joblib
3

Visualization Layer

GSAP scroll narrative reads a single JSON contract. The frontend knows nothing about Python — it only reads predictions.json.

notifier.py → predictions.json
